我喜欢她看似简单的外表，虽然她老是在别家浏览器上“表现不一”。但只要好好“收拾”一番，美丽友好的外貌便又看见了。在使用 CSS 方面，我深谙规范书写 class 和 id 名所带来的美感。在 class 与 id 的连接符应该用“-”，而不是“_” or 驼峰写法。一些细节也显得尤为重要，例如 onfocus="this.blur()" 所带来的读屏障碍。
