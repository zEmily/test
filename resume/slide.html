Fade Example

This is a JavaScript Fade example, using setTimeout to change the CSS opacity of elements. This should work on most objects, including images!

You can use this to rotate your news stories, and as such save some precious space. Just be sure to include a "Show all stories" link, to be nice to your users.

About this Example
